# SDR_Setup
Tools for receiving, transmitting, and decoding HF and other frequencies

Tools Contained in this repo:
GQRX
WSJT-X and dependencies
RTL-SDR Drivers
Audio apps


Installing:
```
git clone https://github.com/cgilbert11/SDR_Setup
```
Then Part1
```
./SDR_Setup_Part1
```
Then Part 2
```
./SDR_Setup_Part2
```
